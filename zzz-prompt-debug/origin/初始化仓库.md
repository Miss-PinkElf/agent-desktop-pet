### 目的
初始化项目结构,我的编程环境是 vscode+wsl，同时需要使用claude code
### 技术栈
#### 前端 frontend
前端初始化之后，记得运行一下看看有没有错误，使用electron官方的npm或者npx命令初始化前端，如果需要输入一些参数，比如项目名称，那么可以在命令后面添加参数，比如
```
npx create-electron-app my-app
```
或者手动输入
1. electron+react+pnpm
2. 使用scss+css module
3. 使用vite
4. 我使用的是wsl，安装依赖的时候使用npm install --platform=win32
#### 后端   backend
先暂时初始化这个虚拟环境的配置，后续再说
1. python，在vscode中开发，简单进行开发，初始化的时候，修改vscode的配置，创建虚拟环境，在创建终端的时候，默认使用虚拟环境，同时里面的环境变量，或者别的，应该和wsl中的一样，比如我可以在wsl中使用claude code的cli，那么在虚拟环境中也可以使用，比如我在虚拟环境中node -v，应该也wsl中一样
2. Python 虚拟环境需要继承 WSL 的环境变量
#### 模型要求
使用 pixi-live2d-display

npm install pixi.js pixi-live2d-display
