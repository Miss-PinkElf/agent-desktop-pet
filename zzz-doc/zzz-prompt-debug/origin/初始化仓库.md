### 目的
初始化项目结构,我的编程环境是 vscode+win，同时需要使用claude code
### 技术栈
#### 前端 
在文件`agent-desktop-pet/frontend`中初始化前端项目
不用建git仓库
前端初始化之后，记得运行一下看看有没有错误，使用electron官方的npm或者npx命令初始化前端，如果需要输入一些参数，比如项目名称，那么可以在命令后面添加参数，比如
```
npx create-electron-app my-app
```
或者手动输入
1. electron+react+pnpm
2. 使用scss+css module
3. 使用vite
#### 后端   backend
在文件`agent-desktop-pet/backend`中初始化后端项目
不用建git仓库
先暂时初始化这个虚拟环境的配置，后续再说
我的项目是前后端在一起了，backend是后端frontend是前端，所以不需要git仓库，需要在项目的根目录配置vscode环境，让终端默认使用  
backend中的虚拟环境，同时运行项目代码的时候也是默认使用backend中的虚拟环境，需要自己创建一个虚拟环境，同时 
1. python，在vscode中开发，简单进行开发，初始化的时候，修改vscode的配置，创建虚拟环境，在创建终端的时候，默认使用虚拟环境，同时里面的环境变量，或者别的，应该和win中的一样，比如我可以在win中使用claude code的cli，那么在虚拟环境中也可以使用，比如我在虚拟环境中node -v，应该也win中一样
2. Python 虚拟环境需要继承 win 的环境变量
